# Warehouse-Automation-with-Autonomous-Robots
This is a fully software-based simulation project focused on automating warehouse operations using autonomous robots. The core of the project is built around ROS (Robot Operating System), which serves as the middleware for controlling and coordinating robots within a simulated warehouse environment.
